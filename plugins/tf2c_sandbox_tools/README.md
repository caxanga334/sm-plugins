# Team Fortress 2 Classified Sandbox Tools

This plugin provides a simple sandbox menu for players.    
It also gives player a simple spawn protection when the sandbox menu is enabled.    

## Configuration

The sandbox menu is only available on specific maps.    
Sandbox maps are configured in `configs/sandbox_maps.cfg`.    
Users with the `sm_sandboxtools_admin` override access can use the menu on any map. The default admin flag is cheats.    

## Usage

Use `sm_sandboxmenu` in the console or `/sandboxmenu` in chat to open the sandbox menu.
